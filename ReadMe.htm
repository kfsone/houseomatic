About HouseOMatic.

HouseOMatic is an EverQuest 2 House Layout file editor. These are local files generated when you access the door to an EQ2 house and select "Save Layout".

While assisting Sadeh of EQ2X's Freeport server, we discovered that loading a layout file can get very slow as you start to fill up a guild hall. More importantly, we found that the layout file isn't absolute: loading a layout file that doesn't match the house contents doesn't cause items to go missing.

This means that if you are working on a small set of items, you can delete everything else from your layout file and speed up the loading process!

Using HouseOMatic.

Saving house layouts.

In order to work with houseOmatic (hOm), you're going to need one or more house layout files.

Saving a layout is easy. Go to the house/guild hall, click the door and select the "Access" menu item. Click the "Save Layout" button, enter a name for the layout, and you're done.

Getting started...

When you fire up hOm you are presented with a simple window with an Open button. I'm honestly not sure why I make you click the Open button to get started, but it seemed like the right thing to do. ;-P

After you click Open, hOm will display a standard Open File dialog. The first time you run hOm, you may need to guide it to the EQ2 folder and more specifically the saved_house_layouts folder.

Of course, this assumes you have any saved layouts. See above.

Once you've loaded a layout into hOm, you'll be presented with a grid view showing you all the items that were saved in the layout.

Layouts don't have to contain all of the items in a house, and houses don't have to contain all the items a layout.

When you load the layout into EQ2, it'll apply changes to whichever items are both in the layout and in the house.
